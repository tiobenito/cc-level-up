The problem

Claude Code can do way more than you think

Skills, hooks, memory, agents, MCP, plugins — there's a lot to discover.

No way to know what's wrong

There's no check engine light for your setup.

No system for learning and improving

You find a tip, want to try it, then forget. This gives you a way to capture, track, and actually adopt new things.

How it works

1
Install the plugin Two commands. Takes 30 seconds.
2
Run /cc-level-up Kicks off a full scan of your setup.
3
Get scanned across 8 areas CLAUDE.md, memory, skills, hooks, MCP, commands, folder structure, context budget. Each graded A through F.
4
See your level (0–4) What's working, what's not, where you rank.
5
Get 2–3 targeted recommendations Not generic tips. Specific actions for YOUR level and YOUR setup.
6
Say yes — it builds the files for you Hooks, skills, configs, memory files. Created and wired up automatically.
ONGOING
7
Capture what you learn with /cc-learn Read an article? Found a tip? Feed it in. It gets saved to your backlog.
8
Run /cc-level-up again See your progress. Get new recommendations. Repeat.

The Level System

0
Getting Started
Just installed. No config. Default everything.
1
Foundation
CLAUDE.md in place. A skill or two. Basic structure.
2
Established
Memory system. References. Several skills. Things are working.
3
Power User
Hooks. MCP. Agents. Context-aware setup. Serious automation.
4
Expert
Learnings loop. Optimized context. Teaches others. Full system.
